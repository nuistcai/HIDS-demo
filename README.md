# HIDS-demo

入侵检测是研究生期间做的方向，为了不留遗憾或者说有个好的总结，我将其开发成一套系统，前端（html）+后端（python）的组合，其中有信息搜集、端口扫描、入侵检测、密码爆破等模块。
